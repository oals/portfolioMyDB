# portfolioMyDB
데이터베이스 조회 어플 모바일 개인 프로젝트 포트폴리오

# 소개
AWS EC2 및 RDS 와 웹 서버를 통해 DB 등록, 데이터 조회가 가능한 안드로이드 어플입니다.
